# Print Studio OpenDesign · 客户话术模板

> 这些话术给使用者直接复制到 Codex / Claude Code 里使用。使用者可以先拖拽图片文件夹，或直接贴图片文件夹路径，然后复制下面任意一段话。

---

## 1. 不知道做什么，先让 Codex 推荐

```text
使用 Print Studio OpenDesign。
我给你一组图片，但还没想好做什么产品。
请先分析这些图片的风格、颜色、主体位置、复杂度和系列统一性，
再推荐适合做什么印刷品、什么尺寸、什么材质和什么版式。
请给出推荐理由，并先生成材质预览和 A/B/C 版式候选。
```

适合：

- 第一次使用。
- 客户只给了图片，没有明确产品。
- 想让 Codex 先做设计判断。

---

## 2. 直接完整跑一遍

```text
使用 Print Studio OpenDesign。
图片路径是：/你的图片文件夹路径
我想先完整跑一遍打样流程。
请自动分析图片、推荐默认规格，生成材质预览、A/B/C 版式候选，
默认先锁定 B 版式，并输出预览图、印刷文件、白底图、氛围图、系列总览图、4K 下载图和交付包。
最后告诉我交付首页在哪里。
```

适合：

- 想快速看到完整结果。
- 演示给客户看。
- 内部测试 Skill 是否可用。

---

## 3. 做 12 个月月历卡

```text
使用 Print Studio OpenDesign。
我要用这组图片做一套 12 张月历卡，年份 2027。
请根据图片推荐尺寸、材质和版式。
先生成材质预览和 A/B/C 版式候选，我确认后再锁版批量输出。
最终需要：预览图、印刷文件、白底图、氛围图、系列总览图、单张 4K 下载图和交付压缩包。
```

适合：

- 插画系列。
- 12 个月月历 / 桌历卡。
- 文具、礼品、节日系列。

---

## 4. 做贺卡

```text
使用 Print Studio OpenDesign。
我要用这组图片做 5×7 英寸贺卡，300g 超白白卡纸，单面，可以圆角。
请先生成 A/B/C 版式候选，并说明每个版式适合什么风格。
我选定后，再锁版生成印刷文件、白底图、氛围图和 4K 下载图。
```

适合：

- 单张或多张贺卡。
- 生日卡、感谢卡、节日卡、邀请卡。

---

## 5. 做明信片

```text
使用 Print Studio OpenDesign。
我要把这组图片做成明信片套装。
如果图片更适合横版，请推荐 7×5 英寸；如果更适合竖版，请推荐 5×7 英寸。
请先生成材质预览和版式候选，并说明推荐理由。
```

适合：

- 风景图、旅行图、插画套装。
- 横版/竖版不确定时。

---

## 6. 做书签

```text
使用 Print Studio OpenDesign。
我要用这组图片做一套书签，尺寸优先考虑 2×6 英寸。
请判断图片是否适合长条排版，如果不适合，请推荐替代尺寸。
需要输出预览图、印刷文件、白底图、氛围图和 4K 下载图。
```

适合：

- 长条插画。
- 植物、诗句、文创书签。

---

## 7. 做吊牌 / 礼品标签

```text
使用 Print Studio OpenDesign。
我要做礼品吊牌，图片是这组素材。
请推荐适合的尺寸、材质和是否需要打孔预留区。
先生成版式候选和印刷预览，注意重要内容不要靠近孔位和裁切边缘。
```

适合：

- 礼盒、产品吊牌、品牌标签。
- 需要打孔或圆角的印刷品。

---

## 8. 做挂历

```text
使用 Print Studio OpenDesign。
我要做 8×12 英寸挂历，12 个月，顶部线圈装订，年份 2027。
请预留顶部装订安全区，重要内容不要靠近打孔/线圈位置。
需要输出预览图、印刷文件、系列总览和交付包。
```

适合：

- 大尺寸展示。
- 线圈挂历、挂孔挂历、夹条挂历。

---

## 9. 只想先看材质

```text
使用 Print Studio OpenDesign。
这组图片还没确定纸张。
请先生成材质预览页，重点比较 300g 超白白卡纸、珠光纸、布纹纸、蓝芯纸、黑芯纸和 PVC。
请说明每种材质适合什么场景，以及小字、浅色图和细节印刷有什么风险。
```

适合：

- 客户在选纸。
- 销售给终端客户看材质方向。
- 不急着出完整排版。

---

## 10. 只想先看版式

```text
使用 Print Studio OpenDesign。
请先不要批量输出完整交付包。
先根据这组图片生成 A/B/C 三个版式候选。
每个候选都要说明适合什么风格、优点和风险。
我选中一版后，你再锁版批量生成。
```

适合：

- 先确认设计方向。
- 不想浪费时间跑完整输出。
- 需要多人评审版式。

---

## 11. 选中某一版后继续

```text
我选择 B 版。
请锁定这个版式，并按这个版式批量生成整套图片。
需要输出预览图、印刷文件、白底图、氛围图、系列总览图、4K 下载图、质检报告和交付压缩包。
```

适合：

- 已经看过 A/B/C 候选。
- 准备进入批量输出。

---

## 12. 要求不要改图 / 不要裁切

```text
使用 Print Studio OpenDesign。
请注意：不要让图像模型直接改成品卡面，不要裁切主体，不要重写文字，不要重排最终卡面。
AI 只做分析和建议，最终排版必须由规则引擎稳定渲染。
白底图和氛围图也要保留完整卡面。
```

适合：

- 客户担心 ChatGPT / Gemini / 即梦改图。
- 强调稳定交付。
- 批量印刷前确认规则。

---

## 13. 要求输出给客户确认

```text
使用 Print Studio OpenDesign。
请生成一份适合发给终端客户确认的交付包。
重点包含：交付说明首页、屏幕预览图、白底图、氛围图、系列总览图和 4K 单张下载图。
印刷文件和质检报告也要保留，但请提醒最终生产以 print/ 文件为准。
```

适合：

- 发终端客户确认。
- 销售报价展示。
- 打样前沟通。

---

## 14. 要求输出给印刷生产

```text
使用 Print Studio OpenDesign。
请生成适合印刷生产检查的交付包。
重点检查：300dpi、出血、安全边距、文字大小、浅色区域、细线装饰、图片分辨率和材质风险。
请输出印刷文件、QC 报告和中文印前说明。
```

适合：

- 交给生产或设计复核。
- 准备打样或小批量印刷。

---

## 15. 一句话最短版

```text
使用 Print Studio OpenDesign。
我给你图片，你先推荐做什么，再生成材质预览、版式候选和完整交付包。
```

适合：

- 新手。
- 快速开始。
- 不想写任何参数。
