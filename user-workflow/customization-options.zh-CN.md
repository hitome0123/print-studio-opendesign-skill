# 可自定义项清单

> 这份清单回答使用者最常问的问题：“我拿到这个 Skill 后，哪些地方可以自己改？”

## 快速可改

| 类别 | 可改内容 | 示例 |
| --- | --- | --- |
| 产品 | 贺卡、明信片、挂历、书签、吊牌、卡牌套装 | `greeting_card` |
| 尺寸 | 预设尺寸或自定义宽高 | `5×7in`、`8×12in` |
| 材质 | 24 种纸张/PVC | `blue_core_300g` |
| 张数 | 单张、4 张、6 张、12 张、自定义 N 张 | `series.count = 8` |
| 标题 | 主标题、副标题、每张标题 | `January`、`HOPE` |
| 输出 | 预览、印刷、白底、氛围、材质页 | `screen`、`print` |

## 专业可改

| 类别 | 可改内容 | 注意 |
| --- | --- | --- |
| 出血 | 默认 3mm，可自定义 | 批量生产前按印厂标准确认 |
| 安全边距 | 默认 5mm，可自定义 | 小字和日期不要贴边 |
| 圆角 | 是否圆角、半径 mm | 需要实际刀模确认 |
| 双面 | 正面 / 背面 | 当前可生成背面设计，生产级双面拼版需另确认 |
| 挂历装订 | 顶部线圈、挂孔、夹条 | 可设置顶部预留 mm |
| 色彩补偿 | 饱和度、对比、纸张补偿 | 非 ICC 真实色彩管理 |

## 设计风格可改

当前稳定风格：

- 复古植物志
- 高级留白卡片
- 温柔插画卡片

可扩展风格：

- 儿童绘本
- 节日礼品
- 轻奢婚礼
- 森系自然
- 极简黑白
- 品牌定制

## 材质预览可改

可模拟：

- 白卡纸
- 牛皮卡纸
- 珠光纸
- 布纹 / 蛋壳纹 / 皱纹
- PVC 光面 / 磨砂
- 蓝芯纸侧边
- 黑芯纸侧边
- 白芯纸侧边

必须说明：

- 材质预览是屏幕模拟。
- 最终颜色和纹理要以实物纸样和打样为准。
- 浅粉、浅黄、浅绿、浅灰字需要特别检查。

## 不建议随便改

- 印刷文件 DPI
- 日期计算规则
- 出血线 / 裁切线逻辑
- QC 规则
- 文件夹输出结构
- 材质 key 名称

这些属于系统规则，建议由开发者或设计负责人维护。
