# Print Studio OpenDesign 客户使用手册

> 这是一套给印刷品打样使用的 Codex Skill。你只需要准备图片文件夹，系统会一步步帮你生成材质预览、版式候选、印刷文件、白底图、氛围图、4K 下载图和交付包。

---

## 1. 这个工具能做什么

Print Studio OpenDesign 适合印刷厂日常接单打样：

- 把一组插画 / 图片做成贺卡、月历卡、明信片、书签、吊牌、挂历等印刷品。
- 根据图片推荐适合的产品、尺寸、纸张材质和排版方向。
- 生成 A/B/C 三种版式候选，选中一版后锁版批量输出。
- 输出屏幕预览图、印刷文件、白底图、氛围图、系列总览图、4K 单张下载图。
- 自动生成质检报告和交付压缩包。

核心原则：

> AI 只负责分析和建议，最终排版由规则引擎稳定渲染，避免聊天模型裁切、改字、重排。

## 2. 最简单的使用方式：拖图片 + 说人话

你不需要先学命令，也不需要编辑 JSON。

最推荐的真实使用方式是：

```text
第 1 步：把图片文件夹拖进 Codex，或者把图片文件夹路径发给 Codex。
第 2 步：用人类语言说你想做什么。
第 3 步：Codex 会自动加载 Print Studio OpenDesign Skill。
第 4 步：Codex 会帮你分析图片、推荐规格、生成预览和交付包。
```

你可以直接这样说：

```text
使用 Print Studio OpenDesign。
这些是我的图片，我想做一套 12 张月历卡。
请先帮我分析图片适合什么尺寸、材质和版式，
再生成材质预览和 A/B/C 版式候选。
```

如果你已经知道要做什么，可以这样说：

```text
使用 Print Studio OpenDesign。
我要用这组图片做 5×7 英寸贺卡，300g 超白白卡纸，单面，圆角。
请直接生成材质预览、版式候选、白底图、氛围图、4K 下载图和印刷文件。
```

如果你什么都还没想好，也可以只说：

```text
使用 Print Studio OpenDesign。
我还没想好做什么，你先根据这些图片帮我推荐。
```

Codex 接下来会用 Skill 自动完成：

- 看图并理解图片风格。
- 推荐产品类型，例如月历卡、贺卡、明信片、书签、吊牌、挂历。
- 推荐尺寸、材质、张数和版式。
- 生成材质预览和 A/B/C 版式候选。
- 你选中一版后，Codex 锁版并批量输出。
- 最后生成交付首页和压缩包。

> 你只需要用自然语言告诉 Codex 想做什么；具体调用哪个脚本、怎么生成文件，由 Codex 和 Skill 完成。

## 4. 如果 Codex 需要你确认，它会问什么

通常 Codex 只会问少量必要问题：

| 问题 | 为什么要问 |
| --- | --- |
| 做什么产品？ | 决定尺寸、版式和输出结构 |
| 做几张？ | 决定单张、套装、12 个月或自定义 N 张 |
| 用什么材质？ | 决定颜色补偿、纹理模拟和印刷风险 |
| 选 A/B/C 哪个版式？ | 锁定后才能稳定批量输出 |
| 是否要印刷文件？ | 如果只做展示图，可以少跑一些输出 |

如果你不确定，可以直接回答：

```text
你先推荐一个默认方案，并说明理由。
```

## 5. 使用前准备

### 5.1 准备图片文件夹

把本次要做的图片放进同一个文件夹，建议这样命名：

```text
01.jpg
02.jpg
03.jpg
...
12.jpg
```

建议：

- 图片尽量按顺序编号。
- 一套系列尽量风格统一。
- 图片分辨率越高越好，批量印刷前需要检查清晰度。
- 如果是 12 个月月历卡，建议准备 12 张图。
- 如果是贺卡 / 明信片 / 书签 / 吊牌，可以是 1 张或自定义 N 张。

### 5.2 如果需要手动运行，再打开运行目录

在 Codex / Claude Code 里进入运行目录：

```bash
cd plugins/print-studio-opendesign/skills/print-studio-runtime
```

如果使用独立 Skill 目录，也可以进入：

```bash
cd print-studio-opendesign
```

## 6. 如果需要手动运行：一键引导流程

第一次使用，推荐直接运行：

```bash
python scripts/run_project_flow.py
```

系统会依次问你：

1. 图片文件夹路径
2. 项目名称
3. 产品类型
4. 系列张数
5. 材质
6. 锁定哪个版式候选

然后它会自动完成：

```text
图片理解
  ↓
材质与尺寸预览
  ↓
A/B/C 版式候选
  ↓
锁定选中的版式
  ↓
批量生成完整交付包
```

### 6.1 如果已经知道参数，可以一条命令跑完

示例：做一套 12 张月历卡，使用 300g 超白白卡纸，锁定 B 版式。

```bash
python scripts/run_project_flow.py \
  --images /你的图片文件夹路径 \
  --theme 小兔子文具主题 \
  --product calendar \
  --count 12 \
  --material 1 \
  --year 2027 \
  --candidate B
```

常用产品类型：

| 参数 | 含义 |
| --- | --- |
| `calendar` | 月历卡 / 桌历卡 |
| `greeting` | 贺卡 |
| `postcard` | 明信片 |
| `bookmark` | 书签 |
| `tag` | 礼品吊牌 |
| `wall_calendar` | 挂历 |

常用材质编号：

| 编号 | 材质 |
| --- | --- |
| `1` | 300g 超白白卡纸 |
| `2` | 300g 冰白珠光纸 |
| `3` | 300g 超白细莱妮纹 |
| `4` | 300g 雅白莱妮纹 |
| `5` | 300g 雅白彩石纹 |
| `6` | 300g 雅白皱纹 |
| `7` | 300g 蓝芯纸 |
| `8` | 330g 黑芯纸 |

也可以直接填写材质 key，例如：

```text
white_card_300g
blue_core_300g
black_core_330g
pvc_30c
```

## 7. 每一步会得到什么

### 第 1 步：图片理解

系统会分析图片：

- 横图 / 竖图 / 方图
- 色彩偏亮还是偏暗
- 饱和度和对比度
- 主体是否居中
- 画面细节是否丰富
- 是否适合做系列

输出文件：

```text
output/项目名/reports/print_studio_advice.md
```

你可以看这份报告来判断：

- 推荐做什么产品
- 推荐什么尺寸
- 推荐什么材质
- 推荐什么排版方向
- 哪些地方需要人工确认

### 第 2 步：材质与尺寸预览

输出文件：

```text
output/项目名/preview/preview.html
output/项目名/preview/materials.html
```

这里可以看：

- 当前尺寸效果
- 当前材质效果
- 24 种纸张 / PVC 材质选择
- 珠光、布纹、蛋壳纹、皱纹、蓝芯、黑芯、PVC 等屏幕模拟

注意：

> 材质预览是屏幕模拟，最终纹理、珠光、PVC 光泽、纸芯效果，以纸样和实物打样为准。

### 第 3 步：A/B/C 版式候选

输出文件：

```text
output/项目名/layout_candidates/index.html
```

你会看到三种候选：

| 候选 | 适合情况 |
| --- | --- |
| A | API / 规则推荐版，比较稳妥 |
| B | 主视觉放大版，适合可爱、细节丰富、想突出插画 |
| C | 高级留白版，适合礼品感、文艺感、克制风格 |

选中一版后，系统会锁定版式。锁版后，后续批量输出会按同一规则渲染，避免每张图风格飘。

### 第 4 步：完整交付包

一键流程结束后，优先打开：

```text
output/项目名/交付说明.html
```

这是总入口，里面会链接到：

- 屏幕预览图
- 印刷文件
- 白底图
- 氛围图
- 系列总览图
- 4K 单张下载页
- 材质预览页
- 版式候选页
- 图片建议报告
- 印前质检报告

## 8. 输出文件怎么理解

完整输出目录类似这样：

```text
output/项目名/
├── 交付说明.html
├── screen/
├── print/
├── commerce/
├── download_4k/
├── preview/
├── layout_candidates/
├── reports/
├── design_plan.json
├── qc_report.json
├── prepress_report.zh-CN.md
└── 项目名_交付包.zip
```

### 8.1 `screen/`

屏幕预览图，用于快速确认画面效果。

### 8.2 `print/`

印刷文件，包含 300dpi、出血和裁切 guide。

最终生产优先看这里。

### 8.3 `commerce/`

展示图，包括：

- 白底图
- 氛围图
- 系列总览图

适合做电商展示、报价沟通、客户确认。

### 8.4 `download_4k/`

单张 4K 下载图，每张可以单独打开和保存。

### 8.5 `reports/`

报告目录，包括：

- 图片建议报告
- 印前质检报告
- 结构化设计计划
- QC 结果

## 9. 如何判断能不能交付

先打开：

```text
output/项目名/交付说明.html
```

再看：

```text
output/项目名/reports/prepress_report.zh-CN.md
```

重点看三类状态：

| 状态 | 含义 |
| --- | --- |
| `PASS` | 当前规则检查通过 |
| `WARN` | 可以继续看效果，但有地方需要人工确认 |
| `FAIL` | 不建议直接交付，需要先修正 |

常见 WARN：

- 图片分辨率偏低
- 小字可能印刷不清楚
- 细线 / 装饰太细
- 浅色区域可能印刷发灰
- 材质纹理可能影响细节

如果是批量印刷，建议最终仍然做一次实物打样。

## 11. 常见问题

### Q1：AI 会不会改图、裁切、改文字？

最终排版不会交给图像模型自由生成。系统会先生成结构化版式计划，再由本地规则引擎渲染，所以可以减少裁切、改字、重排的问题。

### Q2：白底图和氛围图可以直接用于印刷吗？

不建议。白底图和氛围图主要用于展示和确认。

最终印刷以：

```text
print/
```

为准。

### Q3：材质预览是不是实物效果？

不是。材质预览是屏幕模拟，用于快速选择和沟通。最终颜色、纹理、珠光和 PVC 光泽以纸样和打样为准。

### Q4：可以自定义尺寸和材质吗？

可以。默认有常用尺寸和材质，也支持在配置里自定义尺寸、出血、安全区、材质 key、系列张数等。

### Q5：一次只能做 12 个月吗？

不是。可以做：

- 单张贺卡
- 4 张季度卡
- 6 张节日套装
- 12 张月历卡
- 自定义 N 张系列
