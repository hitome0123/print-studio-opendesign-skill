# Print Studio OpenDesign · 图片优先打样建议

## 1. 图片初步判断

- 图片数量：12 张
- 构图方向：方形/接近方形为主
- 色彩气质：柔和浅色，偏暖
- 画面复杂度：细节适中
- 主体位置：主体居中，主体较满
- 系列判断：适合做完整系列

## 2. 建议先问的选择

### 产品方向

1. **5×5 英寸方卡**（默认推荐）
   - 理由：图片接近方形，方卡会更有礼品感和套装感。
2. **5×7 英寸贺卡**
   - 理由：如果需要更多文字空间，可以使用竖版卡并保留上下留白。
3. **12 张月历卡**
   - 理由：图片数量足够做完整月份系列，方形图也适合做上方插画、下方日历的结构。
   - 注意：需要确认是否要真实日期，以及月份文字放在何处。
4. **12 张自定义套装**
   - 理由：素材数量已经具备系列化基础，可以先按现有张数做一套。

### 尺寸方向

1. **5×5 英寸**（默认推荐）
   - 理由：匹配方形构图，适合可爱、艺术、礼品感系列。
2. **5×7 英寸**
   - 理由：需要标题和文案时更好排。

### 材质方向

1. **300g 超白白卡纸**（默认推荐）
   - 理由：最稳定，能保持图片干净明亮，适合首轮打样。
2. **300g 冰白珠光纸**
   - 理由：图片偏柔和浅色，珠光能增加礼品感和梦幻感。
   - 注意：小字要加深，浅色区域要检查发灰。
3. **300g 雅白细莱妮纹**
   - 理由：适合温柔、复古、高级留白方向。
   - 注意：纹理会削弱极细线条。
4. **不优先选牛皮纸**
   - 理由：这组图偏浅，牛皮底色可能让画面变灰变脏。
   - 注意：除非明确想要复古手作感。

### 排版方向

1. **大插画满版 + 小标题**（默认推荐）
   - 理由：主体占比较满，应该突出图本身，减少文字干扰。
2. **底部窄信息区**
   - 理由：如果必须放标题/日期，建议只留底部一条清楚信息区。
   - 注意：避免文字压在复杂画面上。
3. **系列总览网格**
   - 理由：多张图需要放在一起看统一性，建议输出系列总览。

### 字体方向

1. **优雅衬线标题 + 低调无衬线小字**（默认推荐）
   - 理由：柔和图片适合轻盈、有礼品感的文字层级。
2. **手写风点缀**
   - 理由：可以增加温度，但只适合标题或短句。
   - 注意：手写小字不能太小，印刷容易糊。

### 首轮模型预览

1. **先跑即梦 / Gemini / GPT-Image2 三模型预览**（默认推荐）
   - 理由：先用 1 张图比较白底图和氛围图稳定性，再决定后续批量使用哪个模型。
   - 注意：默认不要全量跑，避免浪费额度；缺 API 时只生成错误说明页。

### 印刷检查重点

1. **印刷版提高饱和度和文字对比**（默认推荐）
   - 理由：图片偏浅，印出来容易更灰，需要比屏幕版更扎实。
2. **保留出血和安全边距**（默认推荐）
   - 理由：所有重要文字和主体不能靠边，避免裁切误差。
3. **展示图与印刷文件分离**（默认推荐）
   - 理由：白底图/氛围图用于确认和展示，生产仍以 print/ 为准。

## 3. 可以直接这样引导

我先看了这组图片，建议我们不要盲选规格，而是按图片特征来定：

- 这组图整体是 **方形/接近方形为主**，所以优先推荐上面的产品和尺寸。
- 色彩属于 **柔和浅色**，材质上要优先考虑能保持干净度和对比的纸张。
- 画面是 **细节适中**，排版时要避免文字压住主体，也要检查缩小后的印刷清晰度。

你可以先从默认推荐里选，也可以告诉我：更想要高级感、可爱感、礼品感，还是成本更友好。
